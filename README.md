API REST construida con Spring Boot para la gestión eficiente de estudiantes.

- Descripción

api-student es un servicio backend diseñado para administrar información de estudiantes mediante una arquitectura RESTful, siguiendo buenas prácticas de desarrollo en Spring Boot.

Este proyecto permite gestionar datos de manera estructurada, segura y escalable, sirviendo como base para sistemas educativos, administrativos o proyectos de aprendizaje.

 Características:
 
-API RESTful clara y consistente
-Operaciones CRUD completas
-Validación de datos de entrada
-Manejo de errores centralizado
-Arquitectura en capas (Clean Architecture ready)
-Fácil integración con frontend o servicios externos
